# cs1066
Welcome. We're glad you're here and interested in this experimental course.

### Get started

On this page, click the green "Use this template" button to create a `cs1066` repository under your GitHub account. In the window that pops up, do the following:

1.  Set the new repository's name to `cs1066`
2.  Set this repository's visibility to "Private"
3.  All other settings are fine

When you click the green "Create repository" button at the bottom of the page, you’ll be taken to the new repository in your GitHub account. On this page:

*  Click the green “<> Code” button
*  Select the "Codespaces" tab
*  Click the green "Create codespace on main" button

 This will launch a codespace, and you can now start following the directions discussed in the class notes.

### What's in this starter repo

Configuration files adapt the launched codespace and VS Code for the Web in the following ways:

*  Installs a Python programming environment
*  Upgrades the Python package manager `pip`
*  Modifies a few settings in VS Code

The utility `grab1066.py`, which you'll use to grab the files you need as you move through the course.
